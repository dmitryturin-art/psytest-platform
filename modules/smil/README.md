## 📚 Описание методики

### Общие сведения

**СМИЛ** (Стандартизированный Многофакторный Метод Исследования Личности) — русскоязычная адаптация MMPI (Minnesota Multiphasic Personality Inventory), разработанная в НИИ им. В.М. Бехтерева под руководством Ф.Б. Собчик.

**Назначение:**
- Исследование личности
- Диагностика психических состояний
- Прогноз поведения в стрессовых ситуациях
- Профессиональный отбор

**Формат:**
- 566 вопросов (полная версия)
- 383 вопроса (сокращённая версия)
- Формат ответов: "Да" / "Нет"
- Время прохождения: 30-60 минут

**Целевая аудитория:**
- Возраст: 16-60 лет
- Образование: не ниже 7 классов
- Отсутствие острых психотических состояний

## 📊 Шкалы СМИЛ

### Контрольные шкалы (5)

| Шкала | Название | Назначение | Диапазон | Критические значения |
|-------|----------|------------|----------|---------------------|
| **L** | Шкала лжи | Оценка искренности, социальной желательности | 0-15 | >7 — недостоверно |
| **F** | Шкала достоверности | Выявление симуляции, аггравации | 0-65 | >28 — недостоверно |
| **K** | Коррекционная шкала | Учёт защитной установки | 0-30 | >19 — защита |
| **?** | Шкала неопределённости | Количество ответов «не знаю» | 0-566 | >30 — недостоверно |
| **QC** | Контрольные пункты | Проверка внимательности | 0-27 | <20 — недостоверно |

**Интерпретация контрольных шкал:**

- **L > 7**: Стремление представить себя в лучшем свете, наивная фальсификация
- **F > 28**: Симуляция, аггравация, случайные ответы, непонимание вопросов
- **K > 19**: Психологическая защита, отрицание проблем, формализм
- **K < 10**: Излишняя откровенность, снижение критичности
- **? > 30**: Протокол недостоверен, слишком много неопределённых ответов
- **QC < 20**: Протокол недостоверен, низкая внимательность

**F-K индекс:**
- **> +11**: Симуляция, преувеличение проблем
- **< -12**: Диссимуляция, сокрытие проблем

### Контрольные вопросы (27 пунктов)

В тесте присутствуют 27 контрольных вопросов для проверки внимательности:

```
14, 33, 48, 63, 66, 69, 121, 123, 133, 151, 168, 182, 184, 197, 200, 205,
266, 275, 293, 334, 349, 350, 462, 464, 474, 542, 551
```

**Текст контрольного вопроса:** "Номер данного пункта следует обвести кружочком."

**Правильный ответ:** «Верно» (Да)

**Назначение:** Проверка внимательности и понимания инструкций испытуемым.

**Реализация:** Метод [`calculateControlScale()`](SmilModule.php:549) подсчитывает количество правильных ответов. Если QC < 20, протокол считается недостоверным.

## 🔧 Техническая реализация

### Структура модуля

```
modules/smil/
├── SmilModule.php              # Оркестратор: вызывает калькуляторы, собирает ResultSection[]
├── metadata.json               # Метаданные теста
├── questions-566-full.json     # Все 566 вопросов (ключи исправлены по Соломину)
├── basic_scales_norms.json     # Нормы Собчик для базовых шкал (M, σ, K-вес)
├── additional-scales.json      # Дополнительные шкалы (метаданные/группировка)
├── additional-scales-norms.json # Нормы для T-преобразования доп. шкал
├── interpretations.json        # Текстовые интерпретации по диапазонам T-баллов
├── Scoring/                    # Калькуляторы (чистые классы)
│   ├── RawScoreCalculator.php
│   ├── TScoreCalculator.php
│   ├── ValidityAssessor.php
│   ├── AdditionalScalesCalculator.php
│   └── keys/mmpi-key-solomin.json
└── README.md                   # Эта документация
```

### Алгоритм расчёта

```
1. Сбор ответов (566 вопросов, Да/Нет)
   ↓
2. Расчёт сырых баллов для каждой шкалы
   - Прямые вопросы: ответ "Да" = 1 балл
   - Обратные вопросы: ответ "Нет" = 1 балл
   ↓
3. Конвертация в T-баллы
   - По таблицам для мужчин/женщин
   - Linear interpolation для нецелых значений
   ↓
4. Применение K-коррекции
   - Шкалы 1, 3, 4, 6, 7, 8, 9, 0
   - Формула: T_corrected = T_raw + (K × fraction)
   ↓
5. Определение уровня
   - Низкий (0-44T)
   - Норма (45-54T)
   - Повышенный (55-64T)
   - Высокий (65-74T)
   - Очень высокий (75T+)
   ↓
6. Генерация интерпретации
   - По уровням для каждой шкалы
   - Сочетания шкал (триады, двойные пики)
   - Тип профиля
   - Код профиля (2 ведущие шкалы)
   ↓
7. Визуализация
   - Профиль (ломаная линия)
   - Таблицы с баллами
   - Цветовая кодировка
```

## 📈 Интерпретация профиля

### Типы профиля

#### Нормостенический
- Все шкалы в пределах 45-55T
- Отсутствие выраженных пиков
- **Интерпретация:** Гармоничная личность, хорошая адаптация

#### Невротический (Триада 1-2-3)
- Повышение шкал 1, 2, 3 (>60T)
- **Интерпретация:** Невротические расстройства, конверсионные симптомы

#### Психотический (Триада 6-7-8)
- Повышение шкал 6, 7, 8 (>65T)
- **Интерпретация:** Психотические тенденции, нарушение мышления

#### Психопатический (Триада 4-8-9)
- Повышение шкал 4, 8, 9 (>65T)
- **Интерпретация:** Антисоциальное поведение, импульсивность

### Код профиля

**Двухзначный код:** Две наиболее повышенные шкалы

Примеры:
- **1-2**: Ипохондрический профиль с депрессией
- **2-7**: Тревожно-депрессивный профиль
- **4-9**: Гипертимный профиль с антисоциальностью
- **6-8**: Паранойяльный профиль

## 🎯 Валидность результатов

### Критерии достоверности

| Показатель | Норма | Недостоверно |
|------------|-------|--------------|
| L | < 7 | > 7 |
| F | < 28 | > 28 |
| K | 10-19 | < 10 или > 19 |
| F-K индекс | -12 до +11 | < -12 или > +11 |
| Пропущено вопросов | < 30 | > 30 |

### Причины недостоверности

1. **Симуляция** (F > 28, F-K > +11)
   - Преувеличение проблем
   - "Крик о помощи"

2. **Диссимуляция** (K > 19, F-K < -12)
   - Сокрытие проблем
   - Социально желательные ответы

3. **Случайные ответы** (F > 35)
   - Непонимание вопросов
   - Усталость

## 📝 Использование

### Запуск теста

```php
use PsyTest\Modules\Smil\SmilModule;

$module = new SmilModule();

// Получить вопросы
$questions = $module->getQuestions();

// После получения ответов
$answers = [1 => true, 2 => false, ...];

// Рассчитать результаты
$results = $module->calculateResults($answers);

// Получить интерпретацию
$interpretation = $module->generateInterpretation($results);

// Сгенерировать HTML-отчёт
$html = $module->renderResults($results);
```

## 🔍 Контроль качества

### Проверка перед использованием

1. **Проверить загрузку вопросов:**
   ```bash
   php -r "require 'vendor/autoload.php'; \$m = new SmilModule(); echo count(\$m->getQuestions());"
   ```
   Ожидаемо: 566

2. **Эталонный расчёт (сверка с psytests.org):**
   ```bash
   vendor/bin/phpunit --filter SmilEndToEnd
   ```
   Ожидаемо: 8 тестов, 281 assertion — все зелёные (raw и T-баллы всех 13 шкал совпадают с эталоном ±2).

3. **Симуляция полного прогона:**
   ```bash
   php bin/simulate-smil-test.php
   ```

### Частые ошибки

| Ошибка | Причина | Решение |
|--------|---------|---------|
| 0 баллов | Ответы не конвертируются | Проверить normalizeAnswers() |
| Нет графика | Chart.js не загружен | Проверить подключение скрипта |
| Уровень на английском | Не вызывается getLevelName() | Проверить рендеринг |
