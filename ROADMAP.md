## Видение продукта

Модульная PHP-платформа психологических тестов для практики клинического психолога.
Стек осознанно сохранён (PHP-монолит + MySQL + Twig + DomPDF): деплой на любой
shared-хостинг, минимум зависимостей, полный контроль над расчётами.

Принципы:
- **Корректность расчётов** — высший приоритет. Верификация по методике Собчик + эталон psytests.org.
- **Модульность** — каждый тест изолирован, общая оболочка и дизайн.
- **Профессиональный клинический инструмент**, не «модный».

## Что отложено (явно вне текущего этапа)

- **AI-интерпретация** — инфраструктура есть (OpenRouter/DeepSeek, таблицы `ai_interpretations`/`payment_transactions`), позже как платная/админская функция.
- **16PF (Кеттелл)** — нет исходников/норм.
- **Тест Люшера** — авторский, лицензия не подтверждена. Вне roadmap пока не будет ясности с правами.
- **ТАТ** — качественный анализ текста; упомянут как сценарий.
- **Парный режим** — ✅ реализован для Лазаруса (2026-07-08): flow двух партнёров по ссылкам, сравнение с perception gaps, авто-обновление страницы Партнёра 1. Контракт `supportsPairMode()` / `comparePairResults()` теперь рабочий. Другие тесты могут подключить pair-режим по тому же шаблону.
- **Подростковые нормы СМИЛ** (формы мальчики/девочки) — после извлечения из Собчик. Архитектура под 4 формы заложена, реализованы 2.
- **Админка, аналитика респондентов, rate limiting** — не входят.
